Class AuthViewModel

public class AuthViewModel
extends AndroidViewModel

    ViewModel encargado de manejar las operaciones de autenticación: login y registro de usuarios mediante AuthRepository.

    Expondrá LiveData para que la UI observe el resultado del login y posibles errores de registro.

      • login

         LiveData<LoginResponse> login(LoginRequest req)

        Realiza la operación de login con los datos proporcionados en LoginRequest.

        Se envía la petición al repositorio y se publica el LoginResponse en loginResult si la llamada fue exitosa, o null en caso de fallo.

        Parameters:
        req - DTO con usuario y contraseña para el login.
        Returns:

        LiveData que emitirá el LoginResponse o null.

      • register

         LiveData<UsuarioDTO> register(UsuarioRequestDTO dto)

        Realiza la operación de registro de un nuevo usuario usando UsuarioRequestDTO.

        Crea un < para esta llamada y:

        • Si la respuesta HTTP es exitosa, emite el UsuarioDTO recibido.
        • Si el servidor devuelve un error (códigos 400, 409, etc.), extrae el mensaje del cuerpo de error y lo publica en registerError, emitiendo null.
        • En caso de fallo de red o excepción, publica el mensaje de excepción en registerError y emite null.
        Parameters:
        dto - DTO con los datos del nuevo usuario a registrar.
        Returns:

        LiveData que emitirá el UsuarioDTO en caso de registro exitoso, o null en caso de fallo.
