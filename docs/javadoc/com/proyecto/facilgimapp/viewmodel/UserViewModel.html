Package com.proyecto.facilgimapp.viewmodel

Class UserViewModel

  • All Implemented Interfaces:
    
public class UserViewModel
extends AndroidViewModel

    ViewModel encargado de gestionar las operaciones relacionadas con usuarios: cargar la lista, crear, actualizar, eliminar usuarios y cambiar contraseñas.

    Expone LiveData para:

    • Lista de usuarios.
    • Resultado de validación de la contraseña actual.
    • Resultado de cambio de contraseña.
    • Indicador de éxito o error en operaciones CRUD de usuario.

         void addUser(UsuarioDTO user)

        Crea un nuevo usuario en el servidor.

        Si la creación es exitosa, actualiza _opSuccess a true y recarga la lista de usuarios. Si falla, extrae el mensaje de error del cuerpo de respuesta o usa un mensaje por defecto, publica en _opError y establece _opSuccess a false.
