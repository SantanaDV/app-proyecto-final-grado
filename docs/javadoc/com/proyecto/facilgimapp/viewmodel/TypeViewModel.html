public class TypeViewModel
extends AndroidViewModel

    ViewModel encargado de gestionar los tipos de entrenamiento.

    Proporciona métodos para cargar la lista de tipos, agregar, actualizar y eliminar un tipo, exponiendo los resultados en LiveData.

      • loadTypes

         void loadTypes()

        Carga todos los tipos de entrenamiento desde el servidor.

        Al recibir la respuesta, actualiza _types con la lista obtenida. En caso de fallo, establece el valor en null.

      • addType

         void addType(String nombre)

        Agrega un nuevo tipo de entrenamiento con el nombre indicado.

        Construye un DTO y envía la solicitud. Si es exitoso, recarga la lista de tipos llamando a loadTypes.

        Parameters:
        nombre - Nombre del nuevo tipo a crear.

      • updateType

         void updateType(int id, String nombre)

        Actualiza el tipo de entrenamiento identificado por id, asignándole el nuevo nombre.

        Construye un DTO con el id y nombre, envía la solicitud y, de ser exitoso, recarga la lista de tipos.

        Parameters:
        id - Identificador del tipo a actualizar.
        nombre - Nuevo nombre para el tipo.

      • deleteType

         void deleteType(int typeId, TypeViewModel.DeletionCallback callback)

        Elimina el tipo de entrenamiento con el identificador typeId.

        Si la eliminación es exitosa, recarga la lista de tipos y notifica éxito mediante onSuccess. En caso de fallo, invoca onFailure con un mensaje de error.

        Parameters:
        typeId - ID del tipo a eliminar.
        callback - Objeto que recibe la notificación de éxito o error.
