Package com.proyecto.facilgimapp.viewmodel

Class WorkoutViewModel

  • All Implemented Interfaces:
    
public class WorkoutViewModel
extends AndroidViewModel

    ViewModel que gestiona la lista de entrenamientos del usuario, así como las operaciones de eliminación, actualización y obtención de relaciones entre entrenamientos y ejercicios.

    Expone LiveData para que la interfaz observe cambios en la lista de entrenamientos y mensajes de error.

      • loadWorkoutsByUserId

         void loadWorkoutsByUserId(int userId)

        Carga del servidor todos los entrenamientos asociados al usuario cuyo ID se recibe.

        Si la llamada es exitosa y retorna al menos un entrenamiento, publica la lista en _workouts. Si la lista está vacía o ocurre un fallo HTTP, se asigna una lista vacía y se publica un mensaje de error.

        Parameters:
        userId - ID del usuario cuyas sesiones de entrenamiento se desean obtener.

      • deleteWorkout

         void deleteWorkout(int workoutId, Runnable onSuccess, Runnable onFailure)

        Elimina el entrenamiento identificado por workoutId en el servidor.

        Si la eliminación es exitosa, ejecuta onSuccess.run(). En caso contrario, publica un mensaje de error en errorMessage y ejecuta onFailure.run().

        Parameters:
        workoutId - ID del entrenamiento a eliminar.
        onSuccess - Runnable que se ejecuta tras eliminar correctamente.
        onFailure - Runnable que se ejecuta si ocurre un error.

      • loadWorkoutRelations

         void loadWorkoutRelations(int workoutId, Consumer<List<EntrenamientoEjercicioDTO>> onResult)

        Carga las relaciones entre el entrenamiento (identificado por workoutId) y sus ejercicios asociados, devolviendo la lista resultante al onResult.

        Si la llamada es exitosa, invoca onResult.accept(listaRelaciones). En caso de fallo, registra el error en el log de depuración.
