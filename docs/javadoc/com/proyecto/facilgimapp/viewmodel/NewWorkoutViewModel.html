Package com.proyecto.facilgimapp.viewmodel

Class NewWorkoutViewModel

  • All Implemented Interfaces:
    
public class NewWorkoutViewModel
extends ViewModel

    ViewModel que mantiene el estado de un nuevo entrenamiento en construcción mientras el usuario rellena el formulario antes de iniciar la sesión.

    Guarda en LiveData:

    • Un EntrenamientoDTO parcial (draft) con los campos ya completados.
    • La fecha seleccionada para el entrenamiento.
    • La lista de IDs de ejercicios seleccionados.
    • La posición del Spinner que indica el tipo de entrenamiento.
