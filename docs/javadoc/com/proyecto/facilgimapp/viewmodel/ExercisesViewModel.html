Package com.proyecto.facilgimapp.viewmodel

Class ExercisesViewModel

  • All Implemented Interfaces:
    
public class ExercisesViewModel
extends AndroidViewModel

    ViewModel encargado de gestionar la carga, actualización y eliminación de ejercicios desde el backend mediante EjercicioRepository.

    Proporciona LiveData para:

    • Listado de ejercicios asociados a un entrenamiento específico.
    • Catálogo completo de ejercicios disponibles.
    Además, ofrece métodos para borrar o crear/actualizar ejercicios, notificando éxito o error mediante callbacks. Autor: Francisco Santana
