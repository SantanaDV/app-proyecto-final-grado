Package com.proyecto.facilgimapp.viewmodel

See: Description

  • Class Summary 
    Class Description
    AuthViewModelViewModel encargado de manejar las operaciones de autenticación: login y registro de usuarios mediante AuthRepository.
    ExercisesViewModelViewModel encargado de gestionar la carga, actualización y eliminación de ejercicios desde el backend mediante EjercicioRepository.
    HomeViewModelViewModel que provee los datos estadísticos e informativos para la pantalla de inicio (Home).
    NewWorkoutViewModelViewModel que mantiene el estado de un nuevo entrenamiento en construcción mientras el usuario rellena el formulario antes de iniciar la sesión.
    SeriesViewModelViewModel que gestiona las operaciones CRUD de series asociadas a un entrenamiento o relación específica.
    TypeViewModelViewModel encargado de gestionar los tipos de entrenamiento.
    UserViewModelViewModel encargado de gestionar las operaciones relacionadas con usuarios: cargar la lista, crear, actualizar, eliminar usuarios y cambiar contraseñas.
    WorkoutDetailViewModelViewModel responsable de cargar y exponer los detalles de un entrenamiento específico y sus relaciones con ejercicios (EntrenamientoEjercicioDTO).
    WorkoutSessionViewModelViewModel encargado de guardar la sesión de entrenamiento completada por el usuario.
    WorkoutViewModelViewModel que gestiona la lista de entrenamientos del usuario, así como las operaciones de eliminación, actualización y obtención de relaciones entre entrenamientos y ejercicios.
  • Interface Summary 
    Interface Description
    DeletionCallbackInterface para recibir callbacks tras intentar eliminar un tipo.
