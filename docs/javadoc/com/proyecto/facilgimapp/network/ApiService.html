public interface ApiService

    Interfaz de servicio API para la comunicación con el backend de la aplicación FacilGimApp. Define los endpoints REST disponibles para gestionar ejercicios, entrenamientos, series, tipos de entrenamiento y usuarios, así como autenticación y comprobación del estado del servidor.

    Utiliza Retrofit para la definición de las llamadas HTTP.

    • Ejercicios: CRUD de ejercicios, búsqueda por nombre, gestión de imágenes y relación con entrenamientos.
    • Entrenamientos: CRUD de entrenamientos, filtrado por fechas, usuario y nombre.
    • Series: CRUD de series asociadas a ejercicios en entrenamientos.
    • Tipos de entrenamiento: CRUD de tipos de entrenamiento.
    • Usuarios: CRUD de usuarios, registro, validación y cambio de contraseña.
    • Autenticación: Login de usuario.
    • Estado del servidor: Consulta de salud del backend.
