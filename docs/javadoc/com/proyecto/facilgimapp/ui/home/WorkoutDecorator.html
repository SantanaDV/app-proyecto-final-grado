public class WorkoutDecorator
 implements DayViewDecorator

    Decorador para el calendario que marca con un punto las fechas en las que hay entrenamientos.

    Convierte una lista de java.time.LocalDate a un conjunto de CalendarDay y, para cada día que coincida, añade un punto de color en la vista del día.

    Francisco Santana
