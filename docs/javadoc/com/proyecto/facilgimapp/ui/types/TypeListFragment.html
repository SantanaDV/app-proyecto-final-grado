public class TypeListFragment
extends Fragment implements TypeAdapter.OnTypeInteractionListener

    Fragment que muestra la lista de tipos de entrenamiento y permite crear, editar o eliminar cada tipo.

    Utiliza un RecyclerView con TypeAdapter para presentar los elementos. Al pulsar el FloatingActionButton se abre un diálogo para añadir un nuevo tipo, y mediante clic o clic largo en cada elemento se editan o eliminan respectivamente.
