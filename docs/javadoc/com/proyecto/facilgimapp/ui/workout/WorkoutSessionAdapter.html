Package com.proyecto.facilgimapp.ui.workout

Class WorkoutSessionAdapter

  • All Implemented Interfaces:
    
public class WorkoutSessionAdapter
extends RecyclerView.Adapter<VH>

    Adaptador que gestiona la sesión de entrenamiento en curso mostrando cada ejercicio y permitiendo al usuario añadir, completar o eliminar series (repeticiones × peso).

    Mantiene un mapa interno de SerieDTO para cada EjercicioDTO y ofrece métodos para verificar si todas las series están completadas y para construir la lista final de EntrenamientoEjercicioDTO que envía el ViewModel.

    Author:

    Francisco Santana

      • allCompleted

         boolean allCompleted()

        Verifica si todas las series de todos los ejercicios están completadas.

        Retorna true solo si cada SerieDTO está marcada como completada y tanto repeticiones como peso son mayores que cero. Trata nulos como cero.

        Returns:

        true si todas las series cumplen la condición, false en caso contrario.

      • getEntrenamientoEjercicioDTOList

         List<EntrenamientoEjercicioDTO> getEntrenamientoEjercicioDTOList()

        Construye la lista de EntrenamientoEjercicioDTO a partir de los datos ingresados en seriesMap y la lista de ejercicios.

        Cada EntrenamientoEjercicioDTO contiene el ejercicio, su lista de series, y un orden incremental a partir de 1. Se omiten ejercicios sin series.

        Returns:

        Lista de EntrenamientoEjercicioDTO lista para enviar al ViewModel.
