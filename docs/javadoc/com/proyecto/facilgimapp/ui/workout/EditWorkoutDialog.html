public class EditWorkoutDialog

    Clase encargada de mostrar un diálogo para crear o editar un entrenamiento.

    El diálogo incluye campos para nombre, descripción, duración, fecha y tipo de entrenamiento, así como un listado de ejercicios con sus series correspondientes. Permite seleccionar fecha mediante un DatePicker y actualizar los datos en un EntrenamientoDTO existente.

      • show

         static void show(Fragment parent, EntrenamientoDTO workout, List<TipoEntrenamientoDTO> tipos, Consumer<EntrenamientoDTO> onSave)

        Muestra un diálogo de edición/creación de entrenamiento.

        Rellena los campos con los datos del EntrenamientoDTO pasado, permite modificar nombre, descripción, duración, fecha (con DatePicker), tipo de entrenamiento y editar las relaciones ejercicio-serie. Al pulsar "Guardar", valida que:

        • El nombre no esté vacío.
        • La fecha esté seleccionada.
        • El tipo de entrenamiento esté elegido.
        • Cada ejercicio tenga al menos una serie.
        • Cada serie tenga repeticiones >0, peso >0 y el checkbox 'completada' marcado.
        Si todas las validaciones pasan, actualiza el workout y llama a onSave.accept(workout).
        Parameters:
        parent - Fragment que invoca el diálogo; se utiliza para inflar vistas y acceder al contexto.
        workout - Objeto EntrenamientoDTO que se va a editar (sus campos se rellenan inicialmente).
        tipos - Lista de TipoEntrenamientoDTO disponible para asignar el tipo de entrenamiento.
        onSave - Función que se ejecuta cuando el usuario confirma la edición; recibe el workout modificado.
