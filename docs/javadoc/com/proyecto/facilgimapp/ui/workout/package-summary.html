  • Class Summary 
    Class Description
    EditExercisesAdapterAdaptador para editar las series de cada ejercicio dentro de un entrenamiento en el diálogo de edición.
    EditWorkoutDialogClase encargada de mostrar un diálogo para crear o editar un entrenamiento.
    NewWorkoutFragmentFragment encargado de crear un nuevo entrenamiento paso a paso.
    RelationAdapterAdaptador que muestra la relación ejercicio–serie dentro de un entrenamiento.
    SeriesEditorHelperClase auxiliar que gestiona la edición dinámica de series dentro de un contenedor LinearLayout.
    WorkoutAdapterAdaptador para mostrar una lista de entrenamientos en un RecyclerView.
    WorkoutDetailFragmentFragment que muestra los detalles de un entrenamiento, incluyendo nombre, fecha, duración, tipo y la lista de ejercicios con sus series.
    WorkoutViewHolderViewHolder que representa cada ejercicio dentro de la sesión de entrenamiento, mostrando su nombre, imagen, lista dinámica de series y opciones de menú.
    WorkoutSessionAdapterAdaptador que gestiona la sesión de entrenamiento en curso mostrando cada ejercicio y permitiendo al usuario añadir, completar o eliminar series (repeticiones × peso).
    WorkoutSessionFragmentFragment que gestiona la ejecución de una sesión de entrenamiento.
    WorkoutsFragmentFragment que muestra la lista de entrenamientos del usuario, permite buscar, crear, editar, ver detalles y eliminar entrenamientos.
