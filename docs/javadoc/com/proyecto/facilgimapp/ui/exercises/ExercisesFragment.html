Class ExercisesFragment

  • All Implemented Interfaces:
    android.content.ComponentCallbacks , android.view.View.OnCreateContextMenuListener , androidx.activity.result.ActivityResultCaller , androidx.lifecycle.HasDefaultViewModelProviderFactory , androidx.lifecycle.LifecycleOwner , androidx.lifecycle.ViewModelStoreOwner , androidx.savedstate.SavedStateRegistryOwner 
    
public class ExercisesFragment
extends Fragment

    Fragment que muestra el catálogo de ejercicios, permitiendo buscarlos, filtrarlos, crearlos, editar y eliminar (solo para administradores). Utiliza un RecyclerView con EjercicioCatalogAdapter para presentar la lista, y un FAB para abrir diálogos de creación o edición de ejercicios con carga de imágenes.

    Soporta selección de imagen mediante un ActivityResultLauncher y gestiona la obtención del nombre real de un URI para mantener la extensión correcta.
