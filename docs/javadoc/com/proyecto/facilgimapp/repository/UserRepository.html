Class UserRepository

  • All Implemented Interfaces:
    
public class UserRepository

    Repositorio encargado de gestionar las operaciones CRUD de usuarios contra la API REST mediante Retrofit. Incluye métodos para listar, crear, actualizar y eliminar usuarios, así como para validar y cambiar contraseñas.

    Author:

    Francisco Santana
